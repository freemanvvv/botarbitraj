## Сводка этапов

```
Phase 0: Инфраструктура (LM Studio, модели, Python-окружение)
    ↓
Phase 1: RAG-ассистент по нормативам + генерация документов ← СЕЙЧАС
    ↓
Phase 2: Сметы и BOQ с расчётом стоимости
    ↓
Phase 3: BIM / IFC — генерация скриптов-черновиков
    ↓
Post: Масштабирование, оптимизация, документация
```
